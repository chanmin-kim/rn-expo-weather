# GPS기반 날씨정보 앱

## 기술스택

- React Native Expo
- React Hook
- OpenWeather API

## UX/UI

### 1. 메인페이지
- GPS 기반 날씨정보 표시

### 2. 단위변환
- 화씨/섭씨 단위 변환
- 단위에 따른 정보 표시

### 3. 새로고침
- 새로고침에 따른 API 재요청

### 4. 로딩화면
- API 응답을 받기 전 로딩표시
